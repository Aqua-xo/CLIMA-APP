# 🌤️ Clima TV Premium

Aplicación web del clima desarrollada con HTML, CSS y JavaScript.  
Permite buscar una ciudad y mostrar su temperatura actual, velocidad del viento e ícono climático usando la API de Open-Meteo.

## 🚀 Características
- Búsqueda de ciudades en tiempo real
- Consumo de API con JavaScript
- Diseño visual inspirado en reportes del tiempo de TV
- Fondo animado con sol, nubes y viento
- Interfaz moderna y dinámica
- Mensajes de carga y error

## 🛠️ Tecnologías usadas
- HTML5
- CSS3
- JavaScript
- Open-Meteo API

## 📁 Estructura del proyecto
- `index.html` → estructura principal
- `css/style.css` → estilos y animaciones
- `js/app.js` → lógica principal
- `js/api.js` → conexión con la API
- `js/ui.js` → renderizado de resultados

## ▶️ Cómo usarlo
1. Abre el proyecto en VS Code
2. Ejecuta `index.html` con Live Server
3. Escribe una ciudad
4. Presiona “Buscar clima”

## 📌 Nota
La búsqueda depende de los resultados de geolocalización de Open-Meteo, por lo que algunas ciudades con nombres ambiguos pueden devolver otra coincidencia.
